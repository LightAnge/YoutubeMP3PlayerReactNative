# YoutubeMP3PlayerReactNative
Simple youtube light client made with React Native

# Require :
- Google api keys (youtube query search)


# Install :

### Modules :
Install modules with :
```bash
  npm install
``` 

### Run :
Run with :
```bash
  npm start
  or 
  expo start
```
